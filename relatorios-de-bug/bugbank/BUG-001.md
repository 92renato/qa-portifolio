### 📝 Descrição

Ao realizar o cadastro de um novo usuário utilizando um endereço de e-mail já existente na base de dados, o sistema permite a criação do cadastro sem exibir qualquer validação ou mensagem de erro. Como resultado, os dados do usuário previamente cadastrado são sobrescritos pelos novos dados informados, ocasionando a perda do histórico e das informações associadas ao usuário original.

---

### 🔁 Passos para reproduzir

1. Acessar https://bugbank.netlify.app
2. Clicar em "Registrar"
2. Preencher o campo "E-mail" com um e-mail já cadastrado no sistema
3. Preencher os demais campos com dados válidos
4. Clicar em "Cadastrar"

---

### ✅ Resultado esperado

Sistema deve exibir umensagem de erro informando que o e-mail informado já possui conta ativa. A nova conta não deve ser criada.

---

### ❌ Resultado obtido

Sistema criou novo usuário, sobrescrevendo o usuário anterior.
